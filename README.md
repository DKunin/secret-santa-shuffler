# Array randomizer

> Input array get result of same array shuffled and paired with itself;

## Usage

```js
  
    var randomizeArray = require('randomizearray');

    var shuff = randomizeArray([1,2,3]);
    console.log(shuff); //[[ '1', 3 ],
                        // [ '2', 1 ],
                        // [ '3', 2 ]]

```
